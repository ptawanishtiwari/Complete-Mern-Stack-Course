1. Introduction to Advanced CSS

Advanced CSS allows developers to create professional, interactive and responsive websites. It provides powerful layout systems and visual effects that improve user experience.

Modern websites heavily depend on Flexbox, Grid, Animations and Media Queries to work properly on different devices.

2. Display Property

The display property controls how an element is rendered.

Value Description
block Takes full width
inline Takes content width only
inline-block Combination of block and inline
none Completely hides element 
div{
    display:block;
}

span{
    display:inline;
}

button{
    display:inline-block;
}

3. Position Property

Position property controls the placement of elements.

Value Description
static Default position
relative Relative to itself
absolute Relative to nearest positioned parent
fixed Fixed on screen
sticky Acts like relative and fixed 
.box{
    position:absolute;
    top:50px;
    left:100px;
}

4. Flexbox

Flexbox is a one-dimensional layout system used for aligning and distributing space among elements. 

.container{
    display:flex;
    justify-content:center;
    align-items:center;
}
Property Purpose
justify-content Horizontal alignment
align-items Vertical alignment
flex-direction Row or Column
gap Spacing between items

5. CSS Grid

Grid is a two-dimensional layout system. 

.container{
    display:grid;
    grid-template-columns:
    1fr 1fr 1fr;

    gap:20px;
}

Grid allows control over rows and columns simultaneously.

6. Transform Property

Transform changes shape, size and position of elements. 

.box:hover{
    transform:scale(1.1);
}

.box:hover{
    transform:rotate(45deg);
}
  • translate()
  • rotate()
  • scale()
  • skew()

7. Transition Property

Transition creates smooth effects between property changes. 

.box{

    background:red;
    transition:0.5s;

}

.box:hover{

    background:green;

}

8. Animation

Animations create movement without JavaScript. 

@keyframes move{

from{
    left:0;
}

to{
    left:200px;
}

}

.box{

animation:move 3s infinite;

}

9. Overflow

Controls content that exceeds container size.

Value Description
visible Default
hidden Hide overflow
scroll Always show scrollbar
auto Scrollbar when needed 
.box{
    overflow:auto;
}

10. Z-Index

Z-index controls stacking order of elements. 

.box1{
    z-index:10;
}

.box2{
    z-index:5;
}

11. Media Queries

Media queries help create responsive websites. 

@media(max-width:768px){

h1{
    font-size:24px;
}

}

Different styles can be applied for mobile, tablet and desktop devices.

12. Responsive Design

Responsive design ensures a website looks good on all devices.

  • Fluid Layouts
  • Flexible Images
  • Media Queries
  • Bootstrap Grid System
  • Mobile First Design
