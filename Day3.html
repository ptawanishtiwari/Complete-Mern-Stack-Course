1. Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. HTML defines the structure and content of a webpage.

HTML is not a programming language. It is a markup language that uses tags to organize content.

Example

<h1>Welcome to HTML</h1>
<p>This is a paragraph.</p>

2. HTML Document Structure

<!DOCTYPE html>
<html>
<head>
    <title>My Website</title>
</head>
<body>
    Content Here
</body>
</html>
TagPurpose
htmlRoot Element
headMetadata
titleBrowser Tab Title
bodyVisible Content

SEO tags help search engines understand webpage content.

11. Accessibility

Accessibility ensures websites can be used by all users including people with disabilities.

  • Use alt attribute in images
  • Use semantic HTML
  • Provide labels for forms
  • Maintain proper heading hierarchy
