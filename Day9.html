1. Introduction to JavaScript

JavaScript is a high-level, interpreted and object-oriented programming language. It is mainly used to make websites interactive and dynamic.

HTML provides structure, CSS provides styling and JavaScript provides behavior. Without JavaScript websites remain static and less interactive.

JavaScript runs directly inside the browser and can also be used on servers using Node.js.

2. History of JavaScript

JavaScript was created by Brendan Eich in 1995 while working at Netscape.

  • Originally named Mocha
  • Later renamed LiveScript
  • Finally renamed JavaScript
  • Standardized as ECMAScript

Today JavaScript is one of the most popular programming languages in the world.

3. Why JavaScript is Used?

  • Create Dynamic Websites
  • Form Validation
  • Interactive User Interfaces
  • Single Page Applications
  • Game Development
  • Backend Development
  • Mobile Application Development

JavaScript allows developers to update content without reloading the page.

6. Internal JavaScript

JavaScript code written inside HTML file using script tag is called Internal JavaScript. 

<script>

console.log("Hello JavaScript");

</script>

7. External JavaScript

JavaScript can also be written in a separate .js file. 

<script src="app.js"></script>

This improves code reusability and maintainability.

8. Variables in JavaScript

Variables are containers used to store data values. 

let name = "Awanish";

let age = 24;

Variables make programs flexible because values can be changed when required.

9. Variable Naming Rules

  • Must start with letter, _ or $
  • Cannot start with a number
  • Cannot use reserved keywords
  • JavaScript is case sensitive
  • Meaningful names should be used
Valid Variables
let studentName;

let _age;

let $salary;
Invalid Variables
let 123name;

let var;

10. var Keyword

var is the old way of declaring variables in JavaScript. 

var city = "Lucknow";
  • Function Scoped
  • Can be Redeclared
  • Can be Reassigned

11. let Keyword

let was introduced in ES6 and is the preferred way to declare variables. 

let age = 24;
  • Block Scoped
  • Cannot be Redeclared
  • Can be Reassigned

12. const Keyword

const is used to declare constant values in JavaScript. Once a value is assigned, it cannot be reassigned. 

const country = "India";

console.log(country);
  • Block Scoped
  • Cannot be Redeclared
  • Cannot be Reassigned
  • Must be initialized during declaration

13. Difference Between var, let and const

Feature var let const
Scope Function Scope Block Scope Block Scope
Redeclare Allowed Not Allowed Not Allowed
Reassign Allowed Allowed Not Allowed
Initialization Required No No Yes

14. JavaScript Data Types

Data Type defines the type of value stored inside a variable. JavaScript supports Primitive and Non-Primitive Data Types.

Categories of Data Types

  • Primitive Data Types
  • Non Primitive Data Types

15. Primitive Data Types

Primitive Data Types store single values and are immutable.

Data Type Example
String "Hello"
Number 100
Boolean true
Undefined undefined
Null null
BigInt 123456789n
Symbol Symbol("id")

16. String Data Type

String is used to store textual data. 

let name = "Awanish";

let city = 'Lucknow';

17. Number Data Type

Number stores integer and decimal values. 

let age = 24;

let salary = 25000.50;

18. Boolean Data Type

Boolean stores only two values: true or false. 

let isStudent = true;

let isLoggedIn = false;

19. Undefined Data Type

When a variable is declared but not assigned a value, JavaScript assigns undefined. 

let salary;

console.log(salary);

20. Null Data Type

Null represents an intentionally empty value. 

let address = null;

21. BigInt Data Type

BigInt is used to store very large integer values. 

let bigNumber =
12345678901234567890123456789n;

22. Symbol Data Type

Symbol creates unique identifiers. 

let id = Symbol("user");

23. Non Primitive Data Types

Non Primitive Data Types can store collections of values.

  • Object
  • Array
  • Function

24. Object Data Type

let student = {

name : "Awanish",

age : 24,

city : "Lucknow"

};

Objects store data in key-value pairs.

25. Array Data Type

let skills = [

"HTML",

"CSS",

"JavaScript"

];

Arrays store multiple values in a single variable.

26. typeof Operator

typeof operator is used to check the data type of a value. 

let name = "Awanish";

console.log(typeof name);

console.log(typeof 100);

console.log(typeof true);

27. Practice Program

let name = "Awanish";

let age = 24;

let isDeveloper = true;

console.log(name);

console.log(age);

console.log(isDeveloper);

console.log(typeof name);

console.log(typeof age);

console.log(typeof isDeveloper);
