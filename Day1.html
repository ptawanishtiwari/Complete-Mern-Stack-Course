1. What is Web Development?

Web Development is the process of creating websites and web applications that run on the internet. It includes designing user interfaces, writing business logic, managing databases, deploying applications and maintaining systems.

Types of Web Development

TypeDescription
FrontendUser Interface visible to users.
BackendBusiness logic and APIs.
DatabaseStores application data.

Examples

  • Google Search
  • Facebook
  • Amazon
  • Netflix
  • Instagram

A complete web application combines all three layers: frontend, backend and database.

Express.js is a lightweight framework built on Node.js. It simplifies route creation, middleware management and API development.

7. Node.js Introduction

Node.js is a JavaScript runtime that allows developers to execute JavaScript outside the browser.

  • Fast Execution using V8 Engine
  • Event Driven Architecture
  • Non Blocking I/O
  • Scalable Applications
console.log("Hello Node.js");

8. How MERN Works Together

  1. User interacts with React UI.
  2. React sends API request.
  3. Express receives request.
  4. Node executes business logic.
  5. MongoDB stores/retrieves data.
  6. Response returns to React.
  7. UI updates dynamically.
