1. Introduction to Bootstrap

Bootstrap is the world's most popular CSS framework used for creating responsive and modern websites quickly.

It was developed by Twitter and provides pre-built CSS classes and JavaScript components.

Instead of writing large amounts of CSS manually, developers can use Bootstrap classes to build websites faster.

2. Why Bootstrap?

  • Fast Development
  • Responsive Design
  • Mobile First Approach
  • Cross Browser Compatibility
  • Ready Made Components
  • Professional UI Design

3. Bootstrap Installation

Bootstrap can be added using CDN. 

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css"
rel="stylesheet">

This allows Bootstrap classes to be used immediately.

4. Bootstrap Container System

Containers are used to wrap page content.

Class Description
.container Fixed Width Container
.container-fluid Full Width Container 
<div class="container">

Content Here

</div>

5. Bootstrap Grid System

Bootstrap uses a 12-column grid system. 

<div class="row">

    <div class="col-md-6">
        Left
    </div>

    <div class="col-md-6">
        Right
    </div>

</div>

Grid helps create responsive layouts.

14. Responsive Design

Responsive Design means a website adapts automatically to different screen sizes.

  • Desktop
  • Laptop
  • Tablet
  • Mobile

A responsive website improves user experience.

15. Bootstrap Breakpoints

Breakpoint Screen Size
sm ≥576px
md ≥768px
lg ≥992px
xl ≥1200px
xxl ≥1400px

16. Media Queries

Media Queries allow different CSS styles for different devices. 

@media(max-width:768px){

h1{
font-size:24px;
}

}
