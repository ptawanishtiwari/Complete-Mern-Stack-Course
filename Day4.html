1. Introduction to CSS

CSS stands for Cascading Style Sheets. It is used to style HTML elements and make websites beautiful and responsive.

Without CSS, websites look plain and unattractive. CSS controls colors, layouts, spacing, animations and responsiveness. 

h1{
 color:green;
}

2. Types of CSS

Type Description
Inline CSS Applied directly on HTML element
Internal CSS Written inside style tag
External CSS Written inside separate CSS file

3. CSS Selectors

Selector Example
Element p { }
Class .box { }
ID #header { }
Universal * { } 
.box{
 background-color:yellow;
}

4. Colors in CSS

Colors can be applied using different methods.

h1{
 color:red;
}

h2{
 color:#16a34a;
}

p{
 color:rgb(255,0,0);
}

5. CSS Units

Unit Meaning
px Pixel
% Percentage
em Relative to Parent
rem Relative to Root Element
vh Viewport Height
vw Viewport Width

6. Background Property

body{
 background-color:#f4f4f4;
}

.hero{
 background-image:url(bg.jpg);
 background-size:cover;
}

7. Border Property

.box{
 border:2px solid green;
 border-radius:10px;
}

8. Width & Height

.box{
 width:300px;
 height:200px;
}

9. Margin

Margin creates space outside an element. 

.box{
 margin:20px;
}

10. Padding

Padding creates space inside an element. 

.box{
 padding:20px;
}

11. CSS Box Model

Every HTML element behaves like a rectangular box. 

+-------------------+
|      Margin       |
| +---------------+ |
| |    Border     | |
| | +-----------+ | |
| | | Padding   | | |
| | | Content   | | |
| | +-----------+ | |
| +---------------+ |
+-------------------+
Part Description
Content Actual Content
Padding Space Inside Border
Border Element Outline
Margin Outer Space
